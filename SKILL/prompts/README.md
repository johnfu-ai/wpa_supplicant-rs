# Prompts

## Purpose

Actionable workflow prompts for specific development tasks. Each prompt provides a step-by-step process for a focused activity.

## Available Prompts

| Prompt | When to Use | Related Agent |
|---|---|---|
| `project-kickoff` | Starting the project or a major feature | Requirements Analyst |
| `requirements-elicit` | Eliciting requirements from the standard | Requirements Analyst |
| `requirements-validate` | Validating requirements quality | Requirements Analyst |
| `architecture-starter` | Starting architecture design | Architecture Strategist |
| `design-starter` | Starting detailed component design | Design Engineer |
| `phase-gate-check` | Validating phase exit criteria before transition | Any |
| `tdd-compile` | Executing a TDD Red-Green-Refactor cycle | TDD Driver |
| `test-validate` | Validating test quality and coverage | Testing Specialist |
| `traceability-builder` | Building and validating traceability links | Any |
| `corrective-action-loop` | Fixing failing tests or CI breaks | TDD Driver |
| `security-review` | Performing a Rust security review | Security Analyst |
