# Component Design: eapol-supp — Supplicant EAPOL / PACP

Per IEEE 1016-2009 | ARC-C-EAPOL-002 (#82)

## Component Identity

| Field | Value |
|---|---|
| **Crate** | `crates/eapol-supp/` |
| **Bounded Context** | Supplicant EAPOL |
| **IEEE Clause** | 8 (Supplicant PAE), 11 (EAPOL) |
| **ADRs** | #73 (ADR-WS-001), #74 (ADR-SM-002), #79 (ADR-EVT-007) |
| **Requirements** | #11–#18 (REQ-F-PAE), #44–#47 (REQ-F-EAPOL), #50–#51 (REQ-NF-PERF) |

## DDD Pattern Classification

| Concept | DDD Pattern | Rust Idiom | Rationale |
|---|---|---|---|
| `SupplicantPae` | Aggregate | `struct<C>` owning state, timers, counters | Transactional consistency boundary for PACP (Cl.8) |
| `PaeState` | Value Object | `enum` (Copy) | Finite PACP state enumeration (Cl.8.3) |
| `EapolFrame` | Value Object | `struct` with `Clone, PartialEq` | Immutable after creation; identity is its bytes (Cl.11) |
| `EapolPacketType` | Value Object | `enum` (Copy) | Finite packet type enumeration (Cl.11) |
| `EapolAnnouncement` | Value Object | `struct` with `Clone` | Parsed EAPOL-Announcement (Cl.12) |
| `SupplicantPaeContext` | Repository (trait) | `trait` | Abstracts I/O for dependency injection (ADR-SM-002) |
| `PaeCounters` | Value Object | `struct` with `Clone, Default` | Diagnostic counters (Cl.8.3) — value-based identity |
| `EapolError` | Domain Event (error) | `thiserror` enum | Error classification per ADR-ERR-005 |

## Struct and Enum Definitions

### Supplicant PAE State Machine (Cl.8)

```rust
/// Supplicant PAE state.
///
/// Per IEEE 802.1X-2020, Clause 8.3.
#[derive(Debug, Clone, Copy, PartialEq, Eq)]
pub enum PaeState {
    /// Disconnected — port is down or disabled.
    Disconnected,
    /// Connecting — EAPOL-Start sent, awaiting EAP-Request/Identity.
    Connecting,
    /// Authenticating — EAP exchange in progress.
    Authenticating,
    /// Authenticated — EAP Success received.
    Authenticated,
    /// Held — authentication failed, waiting for heldWhile timer.
    Held,
    /// ForceAuth — administratively forced authenticated.
    ForceAuth,
    /// ForceUnauth — administratively forced unauthenticated.
    ForceUnauth,
    /// Logoff — EAPOL-Logoff sent.
    Logoff,
}

/// Diagnostic counters for the Supplicant PAE.
///
/// Per IEEE 802.1X-2020, Clause 8.3.
#[derive(Debug, Clone, Default)]
pub struct PaeCounters {
    /// EAPOL frames received.
    pub eapol_frames_rx: u64,
    /// EAPOL frames transmitted.
    pub eapol_frames_tx: u64,
    /// EAPOL-Start frames transmitted.
    pub eapol_start_tx: u64,
    /// EAPOL-Logoff frames transmitted.
    pub eapol_logoff_tx: u64,
    /// EAP Response/Identity frames transmitted.
    pub eap_resp_identity_tx: u64,
    /// Last EAPOL frame version received.
    pub last_eapol_version: u8,
}
```

### Supplicant PAE Context Trait (Dependency Injection)

```rust
/// Context trait for Supplicant PAE — abstracts I/O and time.
///
/// Per ADR-SM-002 (#74).
/// Enables mock injection for unit testing.
pub trait SupplicantPaeContext: Send + Sync {
    /// Send an EAPOL frame on the Uncontrolled Port.
    fn send_eapol(&self, frame: &EapolFrame) -> Result<(), EapolError>;

    /// Get the current port state.
    fn get_port_state(&self) -> pae::PortState;

    /// Get the current time (for timer calculations).
    fn now(&self) -> Duration;

    /// Get EAP Response/Identity payload (user identity string).
    fn get_identity(&self) -> &[u8];

    /// Get the configured maximum reauthentication retries.
    fn get_max_retries(&self) -> u32;

    /// Get the heldWhile timer duration (default 60s per Cl.8.3).
    fn get_held_while(&self) -> Duration;

    /// Get the startWhen timer duration (default 30s per Cl.8.3).
    fn get_start_when(&self) -> Duration;

    /// Get the authWhile timer duration (default 30s per Cl.8.3).
    fn get_auth_while(&self) -> Duration;
}
```

### Supplicant PAE Aggregate

```rust
/// Supplicant PAE state machine — Aggregate root for PACP.
///
/// Per IEEE 802.1X-2020, Clause 8.3.
/// Generic over context trait for testability.
/// Owns state, timers, and counters; enforces transition invariants.
pub struct SupplicantPae<C: SupplicantPaeContext> {
    /// Current PACP state.
    state: PaeState,
    /// Retry counter for EAPOL-Start retransmissions.
    start_count: u32,
    /// Maximum start retries before giving up.
    max_retries: u32,
    /// Diagnostic counters.
    counters: PaeCounters,
    /// Timer wheel (uses pae::TimerWheel internally).
    timers: pae::TimerWheel,
    /// Context (injected).
    ctx: C,
}

impl<C: SupplicantPaeContext> SupplicantPae<C> {
    /// Initialize the Supplicant PAE in Disconnected state. Per Cl.8.3.
    pub fn new(ctx: C) -> Self;

    /// Process a received EAPOL frame. Per Cl.8.3.
    ///
    /// Returns events generated by processing.
    ///
    /// # Errors
    /// Returns `EapolError` on frame parse failure or invalid transition.
    pub fn handle_eapol(&mut self, frame: &EapolFrame) -> Result<Vec<pae::PaeEvent>, EapolError>;

    /// Perform a single timer-driven step. Per Cl.8.3.
    ///
    /// Called by the event loop on each tick.
    /// Returns events generated (e.g., EAPOL-Start retransmit, state timeout).
    pub fn step(&mut self) -> Result<Vec<pae::PaeEvent>, EapolError>;

    /// Current PACP state.
    pub fn state(&self) -> PaeState;

    /// Diagnostic counters (read-only).
    pub fn counters(&self) -> &PaeCounters;

    /// Trigger logoff. Transitions to Logoff state. Per Cl.8.3.
    pub fn logoff(&mut self) -> Result<Vec<pae::PaeEvent>, EapolError>;

    /// Trigger reauthentication. Per Cl.8.3.
    ///
    /// Resets state to Connecting, retransmits EAPOL-Start.
    pub fn reauthenticate(&mut self) -> Result<Vec<pae::PaeEvent>, EapolError>;

    /// Handle link state change. Per Cl.8.3.
    ///
    /// If link goes down, transitions to Disconnected.
    /// If link comes up, transitions to Connecting.
    pub fn link_changed(&mut self, up: bool) -> Result<Vec<pae::PaeEvent>, EapolError>;
}
```

### EAPOL Frame Types (Cl.11)

```rust
/// EAPOL protocol version.
///
/// Per IEEE 802.1X-2020, Clause 11.
#[derive(Debug, Clone, Copy, PartialEq, Eq)]
pub enum EapolVersion {
    /// 802.1X-2001 (version 1).
    V1 = 1,
    /// 802.1X-2004 (version 2).
    V2 = 2,
    /// 802.1X-2010 (version 3).
    V3 = 3,
}

/// EAPOL packet type.
///
/// Per IEEE 802.1X-2020, Clause 11.
#[derive(Debug, Clone, Copy, PartialEq, Eq)]
pub enum EapolPacketType {
    /// EAP Packet (0x00).
    EapPacket,
    /// EAPOL-Start (0x01).
    EapolStart,
    /// EAPOL-Logoff (0x02).
    EapolLogoff,
    /// EAPOL-Key (0x03).
    EapolKey,
    /// EAPOL-Encapsulated-ASF-Alert (0x04).
    AsfAlert,
    /// EAPOL-MKA (0x05).
    EapolMka,
    /// EAPOL-Announcement (0x06).
    EapolAnnouncement,
    /// EAPOL-Announcement-Req (0x07).
    EapolAnnouncementReq,
}

/// EAPOL frame — Value Object for frame encoding/decoding.
///
/// Per IEEE 802.1X-2020, Clause 11.
#[derive(Debug, Clone, PartialEq, Eq)]
pub struct EapolFrame {
    /// Protocol version.
    pub version: EapolVersion,
    /// Packet type.
    pub packet_type: EapolPacketType,
    /// Frame body (payload after EAPOL header).
    pub body: Vec<u8>,
}

impl EapolFrame {
    /// EAPOL header size: version(1) + type(1) + length(2) = 4 bytes.
    pub const HEADER_SIZE: usize = 4;

    /// Maximum EAPOL frame body size (per Cl.11).
    pub const MAX_BODY_SIZE: usize = 1500;

    /// Encode the frame to bytes for transmission. Per Cl.11.
    pub fn encode(&self) -> Result<Vec<u8>, EapolError>;

    /// Decode a frame from raw bytes. Per Cl.11.
    ///
    /// # Errors
    /// Returns `EapolError::InvalidFrame` on malformed input.
    pub fn decode(bytes: &[u8]) -> Result<Self, EapolError>;

    /// Create an EAPOL-Start frame. Per Cl.11.
    pub fn start(nid: Option<&[u8]>) -> Self;

    /// Create an EAPOL-Logoff frame. Per Cl.11.
    pub fn logoff() -> Self;

    /// Create an EAPOL-Announcement-Req frame. Per Cl.12.
    pub fn announcement_req() -> Self;

    /// Create an EAP Packet frame. Per Cl.11.
    pub fn eap_packet(eap_data: Vec<u8>) -> Self;

    /// Create an EAPOL-MKA frame. Per Cl.11.
    pub fn mka(mkpdu: Vec<u8>) -> Self;
}
```

## Error Types

```rust
/// Errors for EAPOL supplicant operations.
///
/// Per ADR-ERR-005 (#77).
#[derive(Debug, thiserror::Error)]
pub enum EapolError {
    /// Invalid EAPOL frame (malformed, truncated, oversized).
    #[error("invalid EAPOL frame: {0}")]
    InvalidFrame(String),

    /// EAPOL send failed (L2 socket error).
    #[error("EAPOL send failed: {0}")]
    SendFailed(String),

    /// Timeout waiting for authenticator response.
    #[error("timeout: {0}")]
    Timeout(String),

    /// Invalid PACP state transition.
    #[error("invalid PACP transition: from {from} to {to}")]
    InvalidTransition { from: String, to: String },

    /// Maximum retransmission retries exceeded.
    #[error("max retries ({0}) exceeded")]
    MaxRetriesExceeded(u32),

    /// PAE core error propagated from `pae` crate.
    #[error("PAE error: {0}")]
    Pae(#[from] pae::PaeError),
}
```

## Invariants

| ID | Invariant | Enforced By |
|---|---|---|
| INV-EAPOL-001 | `handle_eapol()` never panics on malformed input | All parse paths return `Result` |
| INV-EAPOL-002 | EAPOL frame body ≤ 1500 bytes | `EapolFrame::decode()` validates |
| INV-EAPOL-003 | Start retransmission count ≤ max_retries | `SupplicantPae::step()` checks |
| INV-EAPOL-004 | PACP transitions follow Cl.8.3 state diagram | `handle_eapol()` and `step()` enforce |
| INV-EAPOL-005 | `step()` returns `Result` — never panics | All paths return `Result<T, EapolError>` |
| INV-EAPOL-006 | EAPOL-Start includes NID if provided | `EapolFrame::start()` encodes NID TLV |
