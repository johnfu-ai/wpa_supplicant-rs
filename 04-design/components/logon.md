# Component Design: logon — Logon Process / NID

Per IEEE 1016-2009 | ARC-C-LOGON-004 (#84)

## Component Identity

| Field | Value |
|---|---|
| **Crate** | `crates/logon/` |
| **Bounded Context** | Logon Process |
| **IEEE Clause** | 12 (Logon Process), 12.5 (NID), 12.6 (CAK Cache) |
| **ADRs** | #73 (ADR-WS-001), #74 (ADR-SM-002), #79 (ADR-EVT-007) |
| **Requirements** | #33–#37 (REQ-F-LOGON) |

## DDD Pattern Classification

| Concept | DDD Pattern | Rust Idiom | Rationale |
|---|---|---|---|
| `LogonProcess` | Aggregate | `struct<C>` owning state, NID groups, CAK cache | Orchestrates PAE, CP, and EAPOL (Cl.12) |
| `LogonState` | Value Object | `enum` (Copy) | Finite Logon Process state enumeration (Cl.12) |
| `NidGroup` | Value Object | `struct` with `Clone, PartialEq, Eq` | Immutable NID definition; identity is its attributes |
| `CakCache` | Entity | `struct` with mutable `HashMap` | Mutable cache with insert/remove/expiry semantics |
| `CakCacheEntry` | Value Object | `struct` with `Clone` | Cached CAK+CKN with metadata |
| `LogonContext` | Repository (trait) | `trait` | Abstracts PAE/CP/EAPOL interactions (ADR-SM-002) |
| `LogonError` | Domain Event (error) | `thiserror` enum | Error classification per ADR-ERR-005 |

## Struct and Enum Definitions

### Logon Process State Machine (Cl.12)

```rust
/// Logon Process state.
///
/// Per IEEE 802.1X-2020, Clause 12.
#[derive(Debug, Clone, Copy, PartialEq, Eq)]
pub enum LogonState {
    /// Initial — no authentication attempted.
    Initial,
    /// NID group selection in progress.
    NidSelection,
    /// Waiting for authentication to complete.
    Waiting,
    /// Authentication succeeded.
    Authenticated,
    /// Authentication failed.
    Failed,
    /// Using unsecured connectivity (fallback).
    Unsecured,
    /// Logoff initiated.
    Logoff,
}

/// Logon Process — Aggregate root for network logon orchestration.
///
/// Per IEEE 802.1X-2020, Clause 12.
/// Orchestrates PAE, CP, and EAPOL interactions.
/// Generic over context trait for testability.
pub struct LogonProcess<C: LogonContext> {
    /// Current Logon state.
    state: LogonState,
    /// Configured NID groups (ordered by preference).
    nid_groups: Vec<NidGroup>,
    /// Currently selected NID group.
    selected_nid: Option<NidGroup>,
    /// CAK cache for pre-shared key acceleration.
    cak_cache: CakCache,
    /// Whether unsecured connectivity fallback is enabled.
    allow_unsecured: bool,
    /// Whether PSK fallback is enabled.
    allow_psk: bool,
    /// Timer wheel.
    timers: pae::TimerWheel,
    /// Context (injected).
    ctx: C,
}
```

### Logon Context Trait (Dependency Injection)

```rust
/// Context trait for Logon Process — abstracts interactions with PAE/CP/EAPOL.
///
/// Per ADR-SM-002 (#74).
/// The Logon Process orchestrates across multiple bounded contexts;
/// this trait provides the integration points.
pub trait LogonContext: Send + Sync {
    /// Start Supplicant PAE authentication. Per Cl.12.
    fn start_authentication(&self, nid: Option<&[u8]>) -> Result<(), LogonError>;

    /// Get current Supplicant PAE state.
    fn pae_state(&self) -> eapol_supp::PaeState;

    /// Get current CP state.
    fn cp_state(&self) -> pae::CpState;

    /// Send EAPOL-Announcement-Req. Per Cl.12.
    fn send_announcement_req(&self) -> Result<(), LogonError>;

    /// Install a pre-shared CAK from cache. Per Cl.12.6.
    fn install_cak(&self, cak: pae::Cak, ckn: pae::Ckn) -> Result<(), LogonError>;

    /// Get current time.
    fn now(&self) -> Duration;
}
```

### NID Group (Cl.12.5)

/// NID (Network Identity) group — identifies a target network.
///
/// Per IEEE 802.1X-2020, Clause 12.5.
#[derive(Debug, Clone, PartialEq, Eq)]
pub struct NidGroup {
    /// NID name (human-readable).
    name: String,
    /// NID identifier bytes (from EAPOL-Announcement).
    id: Vec<u8>,
    /// Associated cipher suite for this NID group.
    cipher_suite: pae::CipherSuite,
    /// Whether PSK is available for this NID group.
    has_psk: bool,
}

impl NidGroup {
    /// Create a NID group. Per Cl.12.5.
    pub fn new(name: String, id: Vec<u8>, cipher_suite: pae::CipherSuite, has_psk: bool) -> Self;

    /// NID name.
    pub fn name(&self) -> &str;

    /// NID identifier bytes.
    pub fn id(&self) -> &[u8];

    /// Cipher suite for this NID group.
    pub fn cipher_suite(&self) -> pae::CipherSuite;

    /// Whether PSK is available for this NID group.
    pub fn has_psk(&self) -> bool;

    /// Match this NID group against an advertised NID from EAPOL-Announcement.
    /// Per Cl.12.5.
    pub fn matches(&self, advertised_nid: &[u8]) -> bool;
}

### CAK Cache (Cl.12.6)

```rust
/// CAK cache entry — a cached CAK+CKN pair with metadata.
///
/// Per IEEE 802.1X-2020, Clause 12.6.
#[derive(Debug, Clone)]
pub struct CakCacheEntry {
    /// Cached CAK.
    cak: pae::Cak,
    /// CKN that identifies this CAK.
    ckn: pae::Ckn,
    /// Cipher suite for this CA.
    cipher_suite: pae::CipherSuite,
    /// Creation time.
    created_at: Duration,
    /// Expiry time (CAK lifetime).
    expires_at: Duration,
}

impl CakCacheEntry {
    /// Create a new cache entry. Per Cl.12.6.
    pub fn new(
        cak: pae::Cak,
        ckn: pae::Ckn,
        cipher_suite: pae::CipherSuite,
        created_at: Duration,
        lifetime: Duration,
    ) -> Self;

    /// Whether this entry has expired.
    pub fn is_expired(&self, now: Duration) -> bool;

    /// The CKN for this cache entry.
    pub fn ckn(&self) -> &pae::Ckn;

    /// The cipher suite for this cache entry.
    pub fn cipher_suite(&self) -> pae::CipherSuite;
}

/// CAK cache — stores pre-shared or previously derived CAKs.
///
/// Per IEEE 802.1X-2020, Clause 12.6.
/// Mutable entity: entries are inserted, looked up, and expired.
pub struct CakCache {
    /// Cached entries indexed by CKN.
    entries: HashMap<pae::Ckn, CakCacheEntry>,
}

impl CakCache {
    /// Create an empty CAK cache.
    pub fn new() -> Self;

    /// Look up a CAK by CKN. Per Cl.12.6.
    ///
    /// Returns `None` if not found or expired (expired entries are removed).
    pub fn lookup(&mut self, ckn: &pae::Ckn, now: Duration) -> Option<&CakCacheEntry>;

    /// Insert a CAK into the cache. Per Cl.12.6.
    pub fn insert(&mut self, entry: CakCacheEntry);

    /// Remove expired entries. Per Cl.12.6.
    pub fn expire(&mut self, now: Duration) -> usize;

    /// Number of cached entries.
    pub fn len(&self) -> usize;

    /// Whether the cache is empty.
    pub fn is_empty(&self) -> bool;
}

impl Default for CakCache {
    fn default() -> Self { Self::new() }
}
```

## Error Types

```rust
/// Errors for Logon Process operations.
///
/// Per ADR-ERR-005 (#77).
#[derive(Debug, thiserror::Error)]
pub enum LogonError {
    /// Invalid NID group.
    #[error("invalid NID group: {0}")]
    InvalidNid(String),

    /// Logon state machine error.
    #[error("logon state machine error: {0}")]
    StateError(String),

    /// No matching NID group found.
    #[error("no matching NID group for advertised: {0:?}")]
    NoMatchingNid(Vec<u8>),

    /// CAK cache miss.
    #[error("CAK cache miss for CKN")]
    CakCacheMiss,

    /// CAK cache expired.
    #[error("CAK cache entry expired")]
    CakCacheExpired,

    /// Authentication failed and no fallback available.
    #[error("authentication failed, no fallback available")]
    NoFallback,

    /// Invalid Logon state transition.
    #[error("invalid logon transition: from {from} to {to}")]
    InvalidTransition { from: String, to: String },

    /// EAPOL error propagated from `eapol-supp` crate.
    #[error("EAPOL error: {0}")]
    Eapol(#[from] eapol_supp::EapolError),

    /// PAE error propagated from `pae` crate.
    #[error("PAE error: {0}")]
    Pae(#[from] pae::PaeError),
}
```

## Invariants

| ID | Invariant | Enforced By |
|---|---|---|
| INV-LOGON-001 | Logon transitions follow Cl.12 state diagram | `handle_*()` methods enforce valid transitions |
| INV-LOGON-002 | NID group selection matches advertised NIDs | `NidGroup::matches()` per Cl.12.5 |
| INV-LOGON-003 | CAK cache entries are expired on lookup | `CakCache::lookup()` checks expiry |
| INV-LOGON-004 | PSK fallback only if `allow_psk` is configured | `LogonProcess::step()` checks flag |
| INV-LOGON-005 | Unsecured fallback only if `allow_unsecured` is configured | `LogonProcess::step()` checks flag |
| INV-LOGON-006 | `step()` never panics — returns `Result` | All paths return `Result<T, LogonError>` |
| INV-LOGON-007 | CAK in cache is zeroized when entry is dropped | `CakCacheEntry` owns `Cak` with `ZeroizeOnDrop` |
